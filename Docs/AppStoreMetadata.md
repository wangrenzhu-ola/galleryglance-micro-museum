# App Store Metadata Draft

## Name
GalleryGlance Micro Museum

## Subtitle
Three-minute art looking ritual

## Promotional Text
Turn a quiet break into a private Eye Note with one artwork card, one visual clue, and one saved observation.

## Description
GalleryGlance Micro Museum helps light art lovers practice looking without turning the moment into a long lesson. Open Today Gallery Card, choose a Color, Composition, or Symbol clue, edit or skip a local prompt helper, then save one private Eye Note to My Micro Museum.

Starter artwork cards are original bundled examples. They are not a live service, community feed, or online art database. All notes and prompt choices stay private on device; no user-approved data leaves the device. Premium Themes are optional StoreKit 2 content packs for additional visual shelves and frame treatments. The core gallery ritual remains free.

## Keywords
art, museum, journal, gallery, notes, mindfulness, visual, education

## Privacy Summary
No login, tracking, backend account, or cloud AI is used in this version. All notes and prompt choices stay private on device; no user-approved data leaves the device.
